Managing Account Information

In this topic you will learn how to manage the Account Information settings.

The Account Information page manages the account name and mailing address. Note that to access the Account Administration page, you must be a user with the Administrator role.

To access the Account Administration page:

  1. Login to the Demand Gen application.
  2. In the navigation header, click the gear icon () and then Account Administration.

Information about the account will be displayed.

To edit your account information click Edit Account Information.

Edit the appropriate values and then click Save. Note that the Account ID cannot be changed.
