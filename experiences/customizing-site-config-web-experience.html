Customizing the Site Configuration of a Web Experience

In this topic you will learn how to customize the site configuration of a web experience.

To configure the Site Configuration settings for a web experience, edit the experience and click SITE CONFIGURATION in the left navigation. The following settings are available:

  • URL - Configure the experience to use a custom domain
  • Access Control - Configure single sign-on and IP restrictions

Using a custom domain

Using a custom domain allows you to host a portal experience on your own URL. For information on using a custom domain, see Assigning a Custom Domain to a Web Experience.

Configuring access control settings

The Access Control settings are used to control who can access a portal experience. Gallery can restrict access using a simple access code, integrate with single sign-on solutions and can restrict access to sites based upon the viewer's IP address. For information on controlling access to Gallery sites, see Controlling Access to a Web Experience.
